# 파일 다운로드 처리
1. 업로드된 자료를 DB나 기타형식으로 웹 화면에 리스트 될 시, 해당 파일을 클릭하면
	서버에 특정위치에 있는 자료를 client(브라우저)를 통해서 현재 pc로 다운로드
	처리하는 것을 말한다.
2. 기본 처리 프로세스
	1) 리스트된 자료를 확인한다.
	2) 클릭시 해당리스트로 자료 요청값 controller(다운로드 전용)를 호출한다.
	3) 해당 컨트롤러에서는 해당 요청으로 되는 파일을 모델값으로 설정한다.
	4) 설정된 모델을 기준으로 다운로드 view를 호출한다.
	5) 클라이언트에 다운로드가 처리된다.
3. 개발 순서
	1) 다운로드 viewer 클래스 작성
		- 모델데이터 : 파일명
		- stream형식을 서버에서 binary 파일을 client에 전달해주는 내용
	2) 컨테이너는 만들어진 viewer 객체를 bean으로 등록 처리
	3) controller 작성
		요청값을 파일명을 받고, 모델데이터를 해당 요청값을 등록 처리
		리턴 문자열을 컨테이너에 등록된 viewer의 id명으로 선언
	4) 테스트
		mvc 패턴으로 만들어진 화면에서 서버에 있는 특정 파일으로 클릭시,
		요청값으로 위에 선언된 controller를 호출 처리 한다.
