# 주요 핵심 키워드와 개념
- Spring Framework : Java 기반의 강력한 엔터프라이즈 애플리케이션
- MyBatis : SQL 매핑 프레임워크로 복잡한 JDBC 코드 작성 없이 데이터 연동을 지원
- 세션(Session) : 서버 측에서 사용자의 상태 정보를 유지하는 방법
- Multipart File : 웹에서 파일을 업로드할 때 사용되는 데이터 형식
- Spring MVC
