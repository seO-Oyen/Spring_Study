# 학습 목표
- Spring Boot 프로젝트의 환경 설정 방법을 이해합니다.
- 초기화면을 구성하는 방법을 학습합니다.
- Spring Boot의 기본 구조와 원리에 대해 배웁니다.

# 주요 핵심 키워드와 개념
- Spring Boot : 자바 기반의 오픈 소스 경량 프레임워크, 빠른 개발을 위한 환경 제공
- Maven/Gradle : 프로젝트 의존성 관리와 빌드를 위한 도구
- Annotations : 코드에 추가적인 정보를 제공 Spring Boot에서 다양한 설정과 기능을 간편하게
	처리
- Controllers : 웹 요청을 처리하는 컴포넌트
- Thymeleaf : 서버 가이드 Java 템플릿 엔진
