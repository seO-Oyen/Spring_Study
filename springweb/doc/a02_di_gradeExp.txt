# DI(dependency injection)
1. 스프링의 컨테이너에 대한 처리를 이해할 수 있는 핵심 부분으로 이에대한 실습은
	스프링의 내부 환경과 처리 플로를 이해할 수 있는 것이다.
2. 아래는 DI를 컨테이너(xml)환경하여서 처리하는 방법을 단계별로 연습할 수 있게 한다.
	1) 단일 객체 선언과 메서드 활용
		- 메서드 활용(property)
		- 생성자 활용(constructor)
	2) 객체와 객체의 관계 설정과 활용
		- 메서드를 통한 객체 선언과 활용
			bean ref="p01"
	3) 여러가지 유형의 객체 활용(collections 객체) 컨테이너에서 활용
		- map
		- list
		- properties
	4) namespace 처리방법의 이해
	5) 의존성 자동 주입처리 연습(Ioc바탕으로 객체간의 자동 주입 처리)
		- 기본 의존성 자동 주입 속성
	6) 옵션에 따른 패키지단위 객체 자동 할당 처리
	7) 패키지단위 객체 자동할당과 소스코드에 직접적 의존성 자동 주의 속성 설정
	8) 외부 설정파일 호출 처리
	9) 실제 컨테이너에 외부 클래스 적용 연습 처리
